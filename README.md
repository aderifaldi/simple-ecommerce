# simple-ecommerce
Simple e-commerce app use mvvm-live and repository pattern

Pattern:
- android livecycle architecture (mvvm, live data)
- repository pattern

Library:
- retrofit (http request)
- glide (image loader)
- butterknife (view binding)
- lombok (model generetor)
- room (local database)

Api:
- bukalapak api (get product)
